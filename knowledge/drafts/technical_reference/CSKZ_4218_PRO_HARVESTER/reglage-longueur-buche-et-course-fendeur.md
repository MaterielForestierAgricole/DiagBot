# Sujet

Coordination du réglage mécanique de longueur et des paramètres de cubage et de course du fendeur sur la KRPAN CSKZ 4218 PRO HARVESTER.

## Objectif

Adapter la longueur des bûches et limiter la course du fendeur afin d'éviter une course inutile lors de la production de bûches courtes.

## Principe

- La longueur physique de coupe est définie par la butée mécanique.
- Le menu 01 règle la longueur utilisée pour le calcul du cubage.
- Le menu 02 sélectionne la course du fendeur parmi trois choix disponibles.

La valeur du menu 01 doit correspondre à la longueur mécanique retenue pour que le cubage soit cohérent. Pour des bûches courtes, sélectionner dans le menu 02 une course adaptée afin de réduire le temps de cycle.

## Procédure

1. Régler la butée mécanique à la longueur de bûche souhaitée.
2. Régler le menu 01 sur la même longueur pour le calcul du cubage.
3. Sélectionner dans le menu 02 la course de fendeur compatible avec cette longueur.
4. Vérifier que la course choisie permet le fendage complet sans déplacement inutile.

## À confirmer

Les trois valeurs disponibles dans le menu 02 et la procédure de navigation dans les menus ne sont pas établies. Les confirmer dans le manuel de commande applicable.
