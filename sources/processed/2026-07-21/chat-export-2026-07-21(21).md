### Assistant

Si le**câble s’enroule mal sur le tambour**du treuil**KRPAN 6,5 DH / 8,5 DH**, il y a plusieurs causes très fréquentes. Voici les points à vérifier dans l’ordre. 🔧

# 1️⃣ Enroulement sans charge (cause la plus fréquente)

Le câble doit toujours être**enroulé sous tension**.Si tu l’enroules sans charge, il se met en vrac et les couches se croisent.

👉 La procédure correcte est :

- **dérouler complètement le câble**,
- puis**le réenrouler en tirant une charge**pour qu’il soit bien serré sur le tambour.6,5 DH FR 8,5 DH FR-1.12.2020

⚠️ Le manuel précise aussi que**les enroulements sans charge de plus de 3 m sont interdits**, car ils provoquent un mauvais empilage du câble.
6,5 DH FR 8,5 DH FR-1.12.2020

# 2️⃣ Mauvais réglage du frein anti-déroulement

Si le**frein anti-déroulement**est mal réglé :

- le tambour tourne trop librement
- le câble se détend
- il s’empile mal.

👉 Le réglage se fait avec la**vis du frein anti-déroulement**.Le bon réglage est atteint quand**le câble peut sortir sans trop d’effort mais reste légèrement freiné**.
6,5 DH FR 8,5 DH FR-1.12.2020

# 3️⃣ Mauvaise première couche sur le tambour

Si la**première couche est mal enroulée**, toutes les autres seront mauvaises.

Solution :

- dérouler presque tout le câble
- laisser**au moins 5 tours sur le tambour**
- réenrouler**lentement sous charge**.6,5 DH FR 8,5 DH FR-1.12.2020

# 4️⃣ Câble vrillé ou qui a fait une boucle

Si une**boucle s’est formée**(lanterne ou torsion) :

- le câble ne s’empile plus correctement
- il saute d’une gorge à l’autre.

Le manuel insiste :➡️ ne jamais laisser le câble faire de boucles lors de l’enroulement ou du déroulement.
6,5 DH FR 8,5 DH FR-1.12.2020

# 5️⃣ Guide / système d’enroulement

Sur le DH il y a un**navijaleur (guide d’enroulement)**qui aide à placer le câble sur le tambour.S’il est :

- grippé
- mal aligné
- ou usé

le câble peut s’empiler d’un seul côté du tambour.

# ⚠️ Astuce terrain (très efficace)

Pour remettre un câble correctement :

- dérouler presque tout
- accrocher le câble à un arbre
- tirer avec le tracteur**en tension constante**
- réenrouler doucement.

C’est la méthode utilisée pour**remettre les couches correctement sur le tambour**.
